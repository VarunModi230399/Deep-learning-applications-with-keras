# Applied Deep Learning with Keras

## Project Overview
This project demonstrates applied deep learning techniques using **Keras with TensorFlow backend**, covering both **computer vision** and **time-series forecasting** use cases. It focuses on building, training, and evaluating neural network models for real-world applications.

## Applications
### 1. Image Classification
- Implemented **transfer learning** using **ResNet50 pre-trained on ImageNet**
- Performed image preprocessing, inference, and top-N class prediction
- Demonstrates practical usage of convolutional neural networks for image recognition

### 2. Stock Price Prediction
- Developed **Recurrent Neural Network (RNN)** models for **time-series forecasting**
- Predicted **Tesla NASDAQ stock prices** using historical market data
- Applied feature scaling, sequential data reshaping, and regression-based prediction
